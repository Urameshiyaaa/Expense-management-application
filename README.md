Các thành viên phải CHECK GITHUB MỖI NGÀY để có thể update code thường xuyên và pull về.
Khi push code lên nhớ PHẢI PUSH VÀO BRANCH của repo để leader check, leader sẽ là người merge code vào main.
Push code lên branch nhớ PHẢI COMMIT để các thành viên nắm bắt được các update mới và tiến độ project.
